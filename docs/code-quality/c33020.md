# Warning C33020

> Likely incorrect HRESULT usage detected.

This warning is a high-confidence indication that an HRESULT-returning function returns FALSE.

## Example

```cpp
#include <Windows.h>

HRESULT foo()
{
    // ......
    return FALSE; // C33020
}
```

These warnings are corrected by using the proper HRESULT value:

```cpp
#include <Windows.h>

HRESULT foo()
{
    // ......
    return E_FAIL; // OK
}
```
